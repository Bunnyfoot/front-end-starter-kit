Welcome to the Front end starter kit

This kit is designed to get your front end projects up and running that bit quicker, while balancing the speed of setup with useful defaults, without bloating the project.

It's not bootstrap or foundation or any other CSS libraries, of mixins for a design you've never seen. This is really truly is a bootstrapper, of just the basics and some placeholders. The rest you add or remove yourself. The idea is that it's just the things used time and again.

Make yourself at home

This kit comes with the following features to give you a boost.

  • Gulp work flow, for those laborious tasks
  • SASS setup for clever, more concise CSS
  • CSS template files to encourage common writing style and best practice
  • A basic style guide for general HTML elements

Style guide

The style guide included, lists bunch of HTML elements that you can style to make sure even on the most basic site elements are going to behave and look as you want them to.

You can adapt the style guide as you see fit, perhaps on large projects using it as a test bed for components. Using partials to manage the SASS for them, loading them only on template pages where they are needed to reduce load.
